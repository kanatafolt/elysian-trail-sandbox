# 重要ルール

- 日本語で簡潔かつ丁寧に回答すること
- 複雑な実装を要求された時は一気に実装せず、小分けに検証可能な単位に切り分けて、進め方の提案を行う

# プロジェクト概要

- Elysian Trailは2Dアニメ風ローグライトRPGの個人開発プロジェクト
- 本プロジェクトではVite+React+TypeScriptを用いたプロトタイプ開発を行う
- UIのベースにはshadcn/uiを用いる
- ページ構成はPCブラウザ・スマホ縦画面の両方に対応させる
- UI・フォントはダークモード・ライトモードに対応させる

# コーディングルール

- コード中のコメントは日本語で記述する

## 基本姿勢

- **インクリメンタルに進める** 大きな変更は小さなステップに分解し、各ステップの動作を確認してから次へ進む
- **確認を取ってから進める** 仕様の解釈に迷いがある場合、実装方針に複数の選択肢がある場合は、作業者に確認を求めてから着手する
- **加算的変更がデフォルト** 「置き換え」と明示されない限り、既存のコードに追加する形で変更する

## 禁止事項

- 指示されていない機能の先回り実装
- 既存コードの大規模リファクタリング（明示的に依頼された場合を除く）
